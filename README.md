# Quantum Symbolic Self-Reading 12T

人生・物語・思想・霊的体験を、`qubit × qubit × qutrit = 2×2×3` の12状態へ構造化して読む、ブラウザ内完結の象徴回路シミュレーターです。

## 位置づけ

12Tは[3Q](https://github.com/AwakeningOS/quantum-symbolic-self-reading-3q)の置き換えではなく、姉妹リポジトリです。

- 3Q: `2×2×2` の8状態モデル
- 12T: `2×2×3` の12状態・混合基数モデル
- QGA: 16状態の二進グリッドモデル

3Qと12Tは主体軸×顕現軸という`2×2`の芯を共有し、`projected_2bit`がその共通射影です。12Tはリズムにおける12系、QGAは16系に対応する姉妹的な実験系として捉えられます。

## 現在の二つの顔

12Tは単に「現在」を足すモデルではありません。

- 内容としての現在: いま起きている接触・選択・揺れ。時間クトリットの`|現在⟩`として表します。
- 様式としての臨在: 時間相同士が切り分けられずに保持される度合い。時間縮約密度行列の非対角コヒーレンスとして読みます。

「現在の人口が大きい」ことと「臨在コヒーレンスが高い」ことは別の発見です。

## 基底

`index = 6*q1 + 3*q2 + t3`

- `q1`: 主体軸（当事/世界、または個我/超越）
- `q2`: 顕現軸（潜在/顕在、または非顕現/顕現）
- `t3`: 時間相（0=過去、1=現在、2=未来）

基底順は`a0,a1,a2,b0,b1,b2,c0,c1,c2,d0,d1,d2`です。3Qでは`a1`が未来でしたが、12Tでは`a1=現在`、`a2=未来`です。両リポジトリのconfigに互換性はありません。

| 当事×顕在 | 来歴 / 遍歴 | 所作 / 行持 | 企図 / 新生 |

各セルは`DEFAULT / SEEKER`の順です。

時間遷移は、過去↔現在を想起・放下、現在↔未来を企投・到来、過去↔未来を跳越と呼びます。

- 主体・顕現・時間の軸別von Neumannエントロピー
- 正規化平均による結びの強さ
- 過去・現在・未来の人口
